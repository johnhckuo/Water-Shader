# Water-Shader

Simulate ocean surface using GLSL shader 

## Installation and Building

```bash
# clone repository
git clone https://github.com/johnhckuo/Water-Shader.git
cd Water-Shader

# install required packages and dependencies
npm install

# build and run locally
npm start
```
